Cartella per gli asset dell'asset store, quando si importano **IMPORTANTE!** spostare gli asset in questa cartella e cancellare il file **nomeasset.meta** dalla cartella asset sennò creerà sempre una cartella vuota per l'asset scaricato.

**STEP QUANDO SI IMPORTA UN ASSET DALL'ASSET STORE**
1) Spostare la cartella **nomeasset** dalla cartella **Assets** alla cartella **ExternalAssets** che si trova al suo interno.
2) Cancellare nella cartella **Assets** il file **nomeasset.meta**.
